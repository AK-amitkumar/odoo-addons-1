Das Gewicht, Nettogewicht und Volumen werden von der Produktvorlage (product.template) zur Produktvariante (product.product) verschoben. Somit kann man diese Werte seperat für jede Produktvariante festlegen.

Das Modul "report_intrastat" benutzt für seine Rechnungen weiterhin die Werte aus den Produktvorlagen.
